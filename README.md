# bs-containers-core

[`OCaml-containers`](https://github.com/c-cube/ocaml-containers) core for BuckleScript.

## Usage

Add `bs-containers-core` to your projects dependencies, Add add it to `bsconfig.json`, e.g.

```json
{
    "name": "coolproj",
    "bs-dependencies": [
      "bs-containers-core",
      ...
    ],
}
```
